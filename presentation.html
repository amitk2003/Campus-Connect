🤖 AI Lost & Found

Smart Matching Engine

📝

TF-IDF Text Similarity

Compares descriptions word-by-word using cosine similarity to find textually matching items.

🖼️

OpenCV Image Hashing

Computes perceptual hash of uploaded images to detect visually similar items, even with minor edits.

🎯

Category & Name Bonus

+15% score for same category; +20% if item names are substrings of each other.

COMBINED SCORE FORMULA

With Images:
Score = Text×0.40 + Image×0.35 + Bonus×0.25

Without Images:
Score = Text×0.70 + Bonus×0.30

MATCH if Score > 0.25

CLAIM PROCESS

1️⃣  Report lost item → AI scans all "Found" reports
2️⃣  Matched items shown ranked by score
3️⃣  Claimer submits proof of ownership
4️⃣  Admin verifies & approves the claim
5️⃣  Item marked Resolved ✓
